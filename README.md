## 基本的数学基础

图像处理的数学方法主要在于近似一个积分泛函（也就是一个“模式”）
$$
\inf \{F(u) = \int_\Omega f(x,u(x),u'(x))dx, u\in V.
$$
这可以用作两个类型的问题：

- 图像恢复
- 序列分割
